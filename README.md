# Manikanta Gonugondla - Portfolio

Modern React portfolio website with fluid animations and unique design.

## Features

- Built with React 18 and Framer Motion
- Fluid, professional animations
- Unique design without generic templates
- Particle network background
- Fully responsive
- Boeing Internship 2026 highlighted

## Installation

```bash
npm install
```

## Development

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
```

Builds the app for production to the `build` folder.

## Technologies

- React 18
- Framer Motion (for fluid animations)
- Custom CSS (no frameworks)
- Canvas API (particle network)
